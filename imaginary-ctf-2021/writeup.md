## 1. Sanity Check - OSINT

### Challenge

Find out the location where he forgot. He still have this picture.
![](https://raw.githubusercontent.com/nh4ttruong/ctf-write-up/main/imaginary-ctf-2021/images/SanityCheck.jpg)

### Solution

First, we find out some special things in the picture. We see the brand sign "SugarPine Bakery".

![](https://raw.githubusercontent.com/nh4ttruong/ctf-write-up/main/imaginary-ctf-2021/images/imagesbrandsign.png)

Use Google Maps Street View to check where "SugarPine Bakery" is.

![](https://raw.githubusercontent.com/nh4ttruong/ctf-write-up/main/imaginary-ctf-2021/images/streetview.png)

Maybe this is the location which we should find out.

### Flag

We have the location **038.9472063,-119.9610895** but the author asked that submit the location with rounded it 3 number.
>**ictf{38.947_-119.961}**
